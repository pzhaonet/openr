《学 R》

  • 零基础学习 R 语言。
  • 手把手带你入门。
  • 研究出版社，北京，2018。

The 'rmd' Package

  • Install the R markdown family.
  • Explore new markdown tools.
  • 一键安装 R markdown 全家桶。
  • 探索 R markdown 工具。

The 'rosr' Package

  • Create a all-in-one project for reproducible research.
  • Even for non-R users.
  • 创建可重复性研究项目。
  • 即使不会用 R 语言，也没关系。

The 'bookdownplus' Package

  • Shortcut to bookdown.
  • Write academic manuscript, thesis, books.
  • 入手 bookdown 的捷径。
  • 书写学术期刊论文，学位论文，书籍。

The 'mindr' Package

  • Draw mind maps with R markdown.
  • Convert between mind maps, R scripts, slides, book chapters...
  • 使用 R markdown 制作思维导图。
  • 在思维导图，R 脚本，幻灯片，书籍章节之间自由转换。

The 'beginr' Package

  • R for beginners.
  • Fundamental applications of R basic functions.
  • 适合 R 初学者。
  • R 基本函数的初级应用。

The 'pinyin' Package

  • Convert Chinese characters into Pinyin, Sijiao or Wubi.
  • Asistance of bookdown.
  • 将汉字转为拼音，四角号码，五笔字型编码。
  • bookdown 的中文助手。

By Peng Zhao on June 23, 2017

Introduction ‘beginr’ is a collection of useful functions for R beginners, including hints for the arguments of the ‘plot()’ function, self-defined functions for error bars, user-customized pair plots and hist plots, enhanced linear regression figures, etc..
